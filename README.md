🧮 Python Basic Calculator

A simple command-line calculator built with Python that performs basic arithmetic operations. This project was created to practice fundamental Python programming concepts such as user input, conditional statements, functions, and arithmetic operations.

# ✨ Features
Addition
Subtraction
Multiplication
Division
Simple command-line interface
Lightweight and beginner-friendly
🛠️ Technologies Used
Python 3

# 📖 How It Works
Run the program.
Enter two numbers.
Select the desired arithmetic operation.
View the calculated result instantly.
🎯 Learning Objectives

Variables and data types

Arithmetic operators
Conditional statements (if, elif, else)
Functions
Basic program structure
